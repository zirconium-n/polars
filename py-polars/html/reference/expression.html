Expressions¶

Functions¶

These functions can be used as expression and sometimes also in eager contexts.

col(name)

A column in a DataFrame.

count([column])

Count the number of values in this column.

to_list(name)

Aggregate to list.

std(column)

Get the standard deviation.

var(column)

Get the variance.

max(column)

Get the maximum value.

min(column)

Get the minimum value.

sum(column)

Get the sum value.

mean(column)

Get the mean value.

avg(column)

Alias for mean.

median(column)

Get the median value.

n_unique(column)

Count unique values.

first(column)

Get the first value.

last(column)

Get the last value.

head(column[, n])

Get the first n rows of an Expression.

tail(column[, n])

Get the last n rows of an Expression.

lit_date(dt)

Converts a Python DateTime to a literal Expression.

lit(value[, dtype])

A literal value.

pearson_corr(a, b)

Compute the pearson’s correlation between two columns.

cov(a, b)

Compute the covariance between two columns/ expressions.

map_binary(a, b, f[, return_dtype])

Map a custom function over two columns and produce a single Series result.

fold(acc, f, exprs)

Accumulate over multiple columns horizontally/ row wise with a left fold.

any(name)

Evaluate columnwise or elementwise with a bitwise OR operation.

all([name])

This function is two things

groups(column)

Syntactic sugar for column(“foo”).agg_groups().

quantile(column, quantile)

Syntactic sugar for column(“foo”).quantile(..).
